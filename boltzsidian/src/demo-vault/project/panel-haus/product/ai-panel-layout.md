# AI panel layout

The feature. User writes a scene description ("street chase, three
beats, ending on a close-up"); the app proposes a page of panels with
rough shapes and suggested action lines. User drags, resizes, accepts
or rejects.

The trick is that the AI isn't drawing; it's _sequencing_. Panel
shape, panel count, reading order, aspect-ratio hints for the
artwork. The actual pencilling is still the user's job. That framing
is what keeps it useful to real manga creators instead of spooky to
them — see [[AI content backlash]].
