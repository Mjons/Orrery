# Cloud library

Every user's panels, pages, characters, and style references are
stored per-account in [[Supabase backend]]. Local-first is nice in
theory; in practice comic artists move between an iPad in the cafe
and a desktop at home, so the cloud is the primary.

Autosave every ~3 s. Version history per page (last 50 autosaves).
Folder organisation free-form.

Trap we've avoided: don't make it a "real cloud drive." It's a
library for _this app's_ content, not a general file system.

Related: [[Export formats]], [[Pro tier]].
