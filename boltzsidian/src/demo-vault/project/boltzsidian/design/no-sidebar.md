# No sidebar

No file tree. No outline panel. No tag list permanently docked to the
left. The universe is the navigation.

If the user wants "show me only /work/," that's a formation (see
[[Formations]]), not a sidebar drill-in. Same end, different mental
model: you don't _leave_ the universe to filter it, you put the
universe through a lens.

This is the structural claim of [[The flat universe ideal]] made
concrete in UI.

Trap: users coming from Obsidian will reach for a sidebar the first
day. The demo vault + coachmarks ([[First-run experience]]) exist to
get them past that reflex.

Related: [[Keyboard first]].
