# Phase 5 — dream mode

One week. The differentiator lands — the app produces a morning
report from a real cycle instead of a canned preview.

In flight:

- Single Sleep Depth float 0..1 drives every parameter regime.
- Idle detector + REM/slow-wave oscillation.
- Shift+D "Dream Now" for manual 60-second cycles.
- `.universe/dreams/YYYY-MM-DD.md` written at wake.
- Prune candidates to `.universe/prune-candidates.json` — never
  auto-deleted.
- Physics softens, chorus quiets + lengthens, camera auto-orbits
  during sleep.

Exit gate: read three morning reports. If none made us notice
anything we hadn't, scoring is over-weighted on recency + degree.
If any read as purple, tone is wrong (see [[Anti-mysticism]]).
