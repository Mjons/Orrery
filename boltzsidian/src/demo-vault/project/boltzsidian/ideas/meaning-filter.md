# Meaning filter

Phase 6 work. The [[Dream mode]] can already produce captions, write
a log, and flag prune candidates. What it can't yet do: score new
ideas that a dream spawns and decide which are worth keeping.

Sketch from BOLTZMANN.md §5.3:

```
M(idea) = novelty × coherence × reach × (1 - age_penalty)
```

Two memories interact during dream; if resonance exceeds θ, a child
idea spawns with a mixed affinity vector. The meaning filter decides
which children promote to files under `ideas/` and which decay
silently.

This is where the K matrix (recorded but unread in [[Phase 3 — physical linking]])
finally earns its keep.

Related: [[Dream mode]], [[Andy Matuschak]] (evergreen notes, but
earned differently).
