# Canvas compatibility

Obsidian Canvas is the feature everyone-except-Michael seems to care
about. Could we import a `.canvas` file as a scene — each canvas card
becomes a body, arrows become tethers, groupings become folders?

Probably yes. The data model is JSON-over-Markdown-paths. An
importer isn't hard.

Open question: does adding this dilute the app's thing? Canvas is
prescribed-layout; Boltzsidian is emergent-layout. Making one a
first-class view of the other could confuse what the tool is _for_.

Shelving until after 1.0.
