# Chorus becoming slop

The failure mode where the [[Observer chorus]] runs and produces
sentences that feel generic, mystical, or just untrue. Easiest way
for this project to lose the user — the chorus is the first thing
they notice after a minute of idling.

Guards in place:

- Every slot in a template is grounded in a specific vault field.
  No slot can resolve to something that isn't in the user's notes.
- Templates are hand-written, tonally flat. See [[Anti-mysticism]].
- Phase 4 exit gate explicitly blocks moving to LLM backends until
  the template pass is already enjoyable.

Warning signs: we reflexively turn it off. We skip it when
demoing. Captions feel mad-libs-generic rather than vault-specific.

Related: [[Phase 4 — observer chorus]], [[Dream becoming mystical]].
