# Scope creep

The speculative docs at the repo root (BRAIN, BOLTZMANN, OBSIDIAN,
DREAM) are great. They're also each a six-month project. The risk is
we try to implement every one of them end-to-end and never ship.

Guard: every phase has an exit gate. Each speculative doc has a kill
condition. Phase 3.5 is [[First-run experience]], not "render a
brain in the negative space." Stay with the user's first five
minutes until those are right, _then_ layer the next speculative
thing.

Currently at: Phase 5. BUILD_PLAN runs to Phase 7. Beta after Phase
5, 1.0 after Phase 6 or 7. Anything after that is a Post-launch
section, not this cycle.
