# Dream becoming mystical

The other failure mode. [[Dream mode]] writes a morning report and
the morning report reads like a horoscope — "the universe noticed
your longing for a deeper truth" etc.

This would be the fastest way to make the product embarrassing. The
Barnum effect guarantees the user will find meaning in any
well-phrased vague statement, so if we lean into that we're building
a very sophisticated fortune cookie.

Guards:

- All template sentences are flat observations, not interpretations.
- We say "noticed," never "realized." "Highest-scoring," never
  "most important."
- Prunings are "worth noticing," not "to cut."
- The dream log is a process log, not a narrative.

See [[Anti-mysticism]] for the founding rule.

Related: [[Chorus becoming slop]], [[Morning report]].
