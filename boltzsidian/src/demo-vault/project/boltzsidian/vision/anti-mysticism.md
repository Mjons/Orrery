# Anti-mysticism

Hard rule from DREAM.md §7. Dream reports written in a mystical voice
will be unreadable by the second day. Captions that aspire to insight
get read generously — the Barnum effect — and the tool becomes a
fortune teller.

Tone has to be flat. "Noticed," never "realized." "Highest-scoring,"
never "most important." Every template sentence we ship is a
prose-level defense of this rule.

If a later LLM backend ([[Phase 5 — dream mode]]'s future upgrade)
starts producing purple text, we fall back to templates. The template
pass is the tonal ground truth.

Related: [[Chorus becoming slop]], [[Dream becoming mystical]],
[[Observer chorus]].
