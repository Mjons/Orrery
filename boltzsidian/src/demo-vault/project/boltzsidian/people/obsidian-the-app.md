# Obsidian (the app)

The prior art. Markdown notes in a local folder, `[[wikilinks]]`, tag
system, graph view. Fundamentally solid product that most serious
notebookers land on.

What we keep:

- Markdown files on disk, authoritative.
- `[[wikilinks]]` syntax, identical semantics.
- Frontmatter conventions (id, created, tags).
- Vault-as-folder.

What we change:

- Graph view becomes the _primary_ surface, not an occasional view.
- No sidebar tree — [[No sidebar]].
- Tags drive appearance, not just filtering — [[Tag vs. folder axis]].
- Add [[Dream mode]] and the [[Observer chorus]], which Obsidian has
  no equivalent of.

Obsidian users can try this without migrating — we read the same files.

Related: [[Andy Matuschak]], [[File System Access]].
