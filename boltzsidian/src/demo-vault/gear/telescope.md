# Telescope

Orion XT8 Dobsonian. 8" (203 mm) primary, f/5.9, 1200 mm focal length.
Manual alt-az mount, no drive, no computer. Bought used 2022.

**What it's good at:** aperture-limited deep-sky. Any bright Messier
object is immediate. Pulls in [[M13 Hercules Cluster|globular cluster]]
resolution at 150× that binoculars will never see.

**What it's bad at:** planetary detail requires better seeing than I
usually get; no tracking, so photo integration is hard without an
equatorial mount. The base is heavy — carrying it up the deck stairs
is a consideration every session.
