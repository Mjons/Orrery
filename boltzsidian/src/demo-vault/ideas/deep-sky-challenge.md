# Deep sky challenge

Ninety deep-sky objects in one calendar year, from the yard. Not
necessarily Messier objects — a mix of Messier, NGC, and a few IC
targets that fit 8"-under-Bortle-6 reality.

**Rules:**

1. Any object seen clearly enough to sketch or photograph counts.
2. One-offs only — no reusing the same object.
3. The object has to be from home or a dark-sky trip I keep notes on.
4. Minimum magnification: the object is "visible," not just in the
   frame.

- [[M45 Pleiades]]
