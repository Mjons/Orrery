# Catalog my sky

My own catalog of what's actually visible on an average Tuesday from
this yard. Not a knockoff of [[Messier]] — a personal, honest
inventory graded for suburban reality.

**Format (draft):**

| field           | notes                            |
| --------------- | -------------------------------- |
| name            | object                           |
| best-ever       | the night I saw it clearly       |
| repeatable      | can I see it any clear night?    |
| requires-travel | or is it a dark-sky-only object? |
| gear            | binos / 8" / DSLR stack          |

**Why:** because every published list assumes either pristine skies
or a professional observatory. Neither is my night. I want the
book I can hand my neighbour, "here's what you'll actually get from
your back steps."

Related: [[Bortle scale]], [[Deep sky challenge]], [[Gear]].
