# Bortle scale

Nine-point scale for night sky brightness, published by John E. Bortle
in 2001 in _Sky & Telescope_. Class 1 is a truly pristine sky;
Class 9 is inner-city.

My yard is Class 6 — suburban sky, Milky Way barely hinted at. The
[[M13 Hercules Cluster]] is a binocular smudge; [[M31 Andromeda]] is
the faintest naked-eye object that works, and only if I know where
to look.

A 90-minute drive east gets me to a Class 3 site. The step change
from 6 to 3 is the single biggest quality-of-life upgrade you can
make as an observer, bigger than any aperture increase.
