# Hotkeys

The canonical list. Press **?** any time in the universe for this same reference as an overlay.

| Key                    | Does                                               |
| ---------------------- | -------------------------------------------------- |
| **N**                  | New note at screen center                          |
| **Cmd+K** / **Ctrl+K** | Search the universe                                |
| **Cmd+D** / **Ctrl+D** | Dream now / morning report                         |
| **L**                  | Cycle label visibility (always / on hover / never) |
| **Shift+F**            | Formations rail                                    |
| **Shift+S**            | Salience debug (power users)                       |
| **\\**                 | Settings                                           |
| **?**                  | This hotkey overlay                                |
| **Esc**                | Close panel / search / modal                       |
| **Alt+drag**           | Create a link between two stars                    |
| **Cmd+Enter**          | Save and close a note                              |
