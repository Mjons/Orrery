# Hotkeys

The canonical list. Press **?** any time in the universe for this same reference as an overlay.

| Key                 | Does                                               |
| ------------------- | -------------------------------------------------- |
| **N**               | New note at screen center                          |
| **E**               | Toggle read / edit mode in the open panel          |
| **Cmd+K** / **/**   | Open search                                        |
| **F**               | Focus the filter bar (top-center)                  |
| **L**               | Cycle label visibility (always / on hover / never) |
| **Shift+F**         | Formations rail                                    |
| **D**               | Morning report from last dream                     |
| **Shift+D**         | Dream now                                          |
| **T**               | Run Tend + open drawer                             |
| **W**               | Open Weed drawer                                   |
| **I**               | Toggle Ideas drawer                                |
| **Cmd+Shift+L**     | Link a keyword to a target note                    |
| **Shift+O**         | Weave — propose cross-links                        |
| **\\**              | Settings                                           |
| **?**               | This hotkey overlay                                |
| **Esc**             | Close panel / search / modal                       |
| **Alt+drag**        | Create a link between two stars                    |
| **Alt+right-click** | Delete a tether under the cursor                   |
| **Cmd+Enter**       | Save and close a note                              |
