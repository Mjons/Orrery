# Boltzsidian

A local-first markdown note-taking app whose workspace is a GPU-accelerated
particle universe.

> Every morning, before coffee, the universe has three things to tell you
> about your own notes.

## Requirements

- Chromium 122+ (Chrome, Edge, Arc, Brave)
- WebGL2 support
- A folder of markdown files (optional — a demo vault ships in Phase 3.5)

## Quickstart

```bash
npm install
npm run dev
```

Open the dev URL, click **Choose workspace folder**, point at any folder of
`.md` files.

## Privacy

Your notes stay on your machine. No telemetry. No phone-home. No cloud sync.
The optional Claude API utterance path (off by default) is the only exception,
and you will see exactly what gets sent before any request.
